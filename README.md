# Анализ макроэкономических данных
Проект реализует обработку CSV файлов с экономическими данными по странам и формирует отчеты. Основной пример отчета - вычисление среднего ВВП по странам. Проект сделан так, чтобы можно было легко добавлять новые виды отчетов в будучщем.

---

## Возможности

* Чтение одного или нескольких CSV-файлов с макроэкономическими данными 
* Формирование отчетов по типу `average-gdp` (Средний ВВП по странам)
* Поддержка расширяемой архитектуры для добавления новых отчетов
* Вывод отччетов на консоль в виде таблицы с помощью `tabulate`
* Легко тестируется с помощью `pytest`

---

## Установка и запуск

### 1. Создать виртуальное окружение 

```powershell
python -m venv .venv
```

### 2. Активировать виртуальное окружение (Windows)

```powershell
.venv/Scripts/activate
```

### 3. Установить зависимости
```powershell
pip install -r requirements.txt
```

### 4. Запустить отчет

```powershell
python main.py --files ./files/economic1.csv ./files/economic2.csv --report average-gdp
```

> CSV-файлы должны содержать колонки `country` и `gdp`

---

## Пример отчёта 

```csv
country, gdp
USA, 100
USA, 300
Germany, 200
Germany, 400
France, 150
```

---

##  Формат отчёта

| Country | Average GDP |
| ------- | ----------- |
| Germany | 300         |
| USA     | 200         |
| France  | 150         |

---

## Тестирование

Запуск всех тестов:

```powershell
python -m pytest
```
